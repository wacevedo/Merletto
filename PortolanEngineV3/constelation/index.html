⌨️ Keyboard Controls

  • V — Switch to vertex mode (for adding/moving points)
  • E — Switch to edge mode (for adding constrained edges)
  • Space — Generate a default grid of points
  • Enter — Reset and clear the canvas
  • C — Export pattern as SVG file
  • G — Export graph (points + constraints) as JSON
  • L — Load graph from JSON file

🖱️ Mouse Controls

  • Click — Add points to the canvas (in vertex mode)
  • Click + Drag — Move existing points (in vertex mode) or create a constrained edge (in edge mode)
  • Shift + Click — Remove a point
